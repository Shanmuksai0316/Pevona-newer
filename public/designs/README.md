# Design Images

This folder contains design mockups and reference images from the Google Drive folder.

## Folder Structure

Place the downloaded images here with descriptive filenames:

- `home-page.png` - Home page design
- `about-us.png` - About us page
- `application-process.png` - Application process page
- `buy-guide-area-guide.png` - Buy Guide & Area Guide
- `client-portal.png` - Client Portal page
- `compliance-licensing.png` - Compliance & Licensing
- `property-search.png` - Finding property & property search
- `hmo.png` - House Of Multiple Occupations (HMO)
- `investment-opportunities.png` - Investment Opportunities
- `landlord.png` - Landlord page
- `maintenance-repair.png` - maintenance & repair
- `market-insights.png` - Market Insights
- `management-services.png` - Our Management Services
- `portfolio.png` - Portfolio page
- `properties-to-let.png` - Properties to let
- `sell.png` - Sell page
- `tenant-services.png` - Tenant services

## How to Download from Google Drive

1. Open the Google Drive folder: https://drive.google.com/drive/folders/1D61PV84VmkxcQYC_unior52Y2VQefA84
2. Select all images you want to download
3. Right-click and choose "Download" or use the download button
4. Extract the downloaded ZIP file
5. Move the images to this `public/designs` folder with the naming convention above

## Usage

These images can be referenced in your Next.js app using:
```jsx
<Image src="/designs/home-page.png" alt="Home Page Design" />
```
